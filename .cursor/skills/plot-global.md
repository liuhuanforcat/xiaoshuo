# 整体剧情结构

## 五卷总览

```
卷一 · 天命之子     [1-50章]    → 觉醒 + 被压制 + 逃离
卷二 · 大荒历练     [51-150章]  → 成长 + 结盟 + 揭秘身世
卷三 · 诸神争霸     [151-300章] → 势力博弈 + 掌纹危机爆发
卷四 · 逆天改命     [301-450章] → 最终对决 + 残玉真相大白
卷五 · 长夜将明     [451-500章] → 收尾 + 新时代开启
```

## 各卷关键事件

### 卷一 · 天命之子
- 残玉觉醒，掌纹之谜浮出水面
- 宗门倾轧，被逐出师门
- **逃亡缓冲带**：顺卓河·桃源村与岱家（人间锚点）→ **普江山洞府**得《虚空步》正法（相对天镜前辈遗笔）
- **岱家劫**：家传**清玄镜**遭**鬼灵宗**觊觎；村灭，**岱伯**战死（碎镜破局），**岱容儿**失踪线，**岱小禾**下游幸存可作伏笔
- **外城成长**：废窑**黑市**（奚岚/冷淬方/沉河砂）→ **淬体**奠基 → **虚空步**二重「穿线」
- **外城冲突**：王都**小王爷**跋扈线（救民女，权斗种子）；**阿沅**（将军遗孀）赠**下品灵剑**（人情）
- **天镜司近身**：客栈女暗探**沈棠**试探；**聚宝斋**拍得**吞金胚**黑石，灵剑被吞，凝为**体内金线小剑**（破隙之锋，仍吃掌纹）
- **南荒河线升级**：赤沙渡黑船→雾门集骨册与“红轿饵局”→水下救回岱容儿真身；确立“半片河眼苗”设定
- **修复与反击**：瘴寨阿婆断线、镜瞎子交易、裂石峡镜匣过境；林玄夺回骨铃“杖珠残渣”，被王都副司标记为**缺口样本**
- **转场秘境前域**：黑石渡无名帖、听鼓石三响开缝，进入北隘带；引路客莫问来路登场
- 九龙秘境开启，踏入更广阔天地（可与上述线索交汇）

### 卷二 · 大荒历练
- 荒野独行，实力急速提升
- 结识霜凝与一众同伴
- 身世线索首次出现

### 卷三 · 诸神争霸
- 各大势力正面冲突
- 掌纹已去六道，危机感拉满
- 司徒渊真实目的暴露

### 卷四 · 逆天改命
- 与司徒渊决战
- 残玉来历揭晓
- 以最后一道掌纹换取胜利

### 卷五 · 长夜将明
- 世界秩序重建
- 掌纹神秘复原，留下开放结局悬念
- 九龙再现，天地迎来新纪元

## 全局悬念钩子

1. 残玉究竟是什么？为何九龙见之俯首？
2. 掌纹归零会死，主角如何破局？
3. 司徒渊与主角之间是否存在更深的联系？
4. 清玄镜碎片与河灵碎片最终将引向秘境、王都还是归墟？
5. 金线小剑与九龙门「第九门」是否同铸一脉？
6. 雾门集骨册被焚后，谁掌握“残名册”与人皮供应链？
7. 沈棠与镜瞎子关系：合作、利用，还是师徒旧线？
