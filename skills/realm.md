# 修炼境界

## 境界总览

```
淬体境 → 聚灵境 → 化元境 → 开脉境 → 铸魂境
    ↓
凝虚境 → 破天境 → 归墟境 → 混沌境 → 无极境（传说）
```

## 境界详情

| 境界 | 层数 | 代表能力 | 世间占比 |
|------|------|----------|----------|
| 淬体境 | 九层 | 强化肉身，抗击打，初步感知灵气 | 90% 的修士止步于此 |
| 聚灵境 | 九层 | 将灵气汇聚丹田，可施展基础术法 | 常见，各大宗门弟子标配 |
| 化元境 | 九层 | 灵气转化为元力，质变，战力倍增 | 宗门核心弟子层级 |
| 开脉境 | 九层 | 打通十二正经奇脉，元力自循环 | 长老级，百里挑一 |
| 铸魂境 | 九层 | 神魂成形，可神识出窍，千里感知 | 宗主级，万中无一 |
| 凝虚境 | 三层 | 触摸虚空法则，可撕裂空间 | 一国之中不超过十人 |
| 破天境 | 三层 | 法则初成，举手投足天地异变 | 大陆级强者，凤毛麟角 |
| 归墟境 | 一层 | 掌控一种天地法则，近乎不死 | 传说中存在，从未现世 |
| 混沌境 | 一层 | 超脱轮回，生死由心，可毁天地 | 史书记载，仅一人到达 |
| 无极境 | — | 与天地同寿，法则即我，我即天地 | 神话，无人证实存在 |

## 林玄境界成长路线

| 章节 | 境界 | 说明 |
|------|------|------|
| 第1章 | 淬体境三层 | 天才眼中的废物，全宗垫底 |
| 第20章 | 化元境二层 | 残玉觉醒后跨越式突破，众人震惊 |
| 第50章 | 开脉境七层 | 秘境历练后，隐隐触摸铸魂边缘 |
| 第150章 | 凝虚境一层 | 身世揭秘当日，强行突破 |
| 第300章 | 破天境二层 | 掌纹剩最后一道，以命换境 |
| 第450章 | 归墟境 | 残玉碎裂，真相大白，完成蜕变 |

## 残玉与境界的隐藏关联

> 每消失一道掌纹，下一境界突破阻力降低一半。
> 九道掌纹恰好对应九次关键突破——残玉本质上是一把**用寿命换速度**的钥匙。
